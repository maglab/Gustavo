# Pleiotropy and Disease Interactors

This repository contains the **analysis code** required to reproduce the results presented in the study  
**“Pleiotropy and Disease Interactors: The Dual Nature of Genes Linking Ageing and Ageing-related Diseases.”**

---

## Repository contents

The repository is organized as follows:

- **Code/**  
  Contains all scripts used for data processing, network analysis, and figure generation.  
  Scripts reference external data locations for loading inputs and saving outputs.

- **README.md**  
  Provides a high-level overview of the project structure and data access.

---

## Data availability

Due to GitHub file size limitations, **datasets are not hosted directly in this repository**.  
All retrieved source data and generated results are available through the **Synapse data-sharing platform** under the project:

**Pleiotropy and disease interactors**  
**Synapse Project ID:** `syn72037936`  
**Direct link:** https://www.synapse.org/#!Synapse:syn72037936

Data are organized in structured subdirectories corresponding to retrieved datasets and generated analysis outputs.

---

## Reproducibility

The scripts provided in this repository are designed to work directly with the data hosted on Synapse.  
Detailed descriptions of the analysis pipeline, individual scripts, and data organization are provided within the code and accompanying documentation.
